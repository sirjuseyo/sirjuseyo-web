- **배경**: `#000` (검정) + 초록/노랑 유기적 도형 장식
- **콘텐츠 순서**:
  1. 타이틀: `내가 지인에게` + `평판을 인정받으면` (노란색/흰색, 볼드)
  2. 질문 pill: ❶ `대출을 받을 수 있다구요?` (흰색 bg, 검정 텍스트)
  3. 브라우저 카드 (맥 창 UI):
     - `맞아요! 지인에게 인정받은` (검정)
     - `내 평판으로도` (보라색 `#7C4DFF`, 볼드)
     - `대출 기회가 열려요.` (검정)
  4. 매니저 프로필 카드:
     - 아바타 아이콘 + `매니저` 🔍
     - `신용점수, 소득, 기대출 때문에` + `계속 대출이 거절되어서 속상했죠?` + `이제는 그런 걱정은 끝!` (볼드)
