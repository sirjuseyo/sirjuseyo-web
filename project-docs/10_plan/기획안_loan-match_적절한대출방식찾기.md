## 6. 구현 참고사항

1. **Tally → HTML 전환**: 원본은 Tally 폼이었으나, 자체 HTML로 제작
2. **조건 로직**: 질문 응답에 따른 카드 표시/숨김은 JavaScript로 구현
3. **질문 없이 바로 카드 노출 여부**: 댄디어빠쮸너야님 확인 필요 — 질문 폼 유지 or 카드만 바로 노출
4. **카드 링크 목적지**: 각 카드 클릭 시 이동할 페이지 추후 확정 필요
5. **loan-match 메인 기획서 업데이트 필요**: SEC-05 버튼 링크를 `../nanocredit/index.html` → `find.html`로 변경
