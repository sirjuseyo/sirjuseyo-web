```css
--brand:    #380097;    /* 써주세요 메인 (PANTONE 2735 XGC) */

--highlight:#FEE500;    /* 노란 말풍선 (클라이맥스) */
```

### 레이아웃
- 모바일 퍼스트, 최대 폭 600px, 중앙 정렬
- 이미지(007~009.png)는 `width: 100%`로 전체 너비 삽입
- 클로징 이미지(006.png)는 최대 400px, 중앙 정렬

### 원본 충실도 우선 UI 컴포넌트
| 컴포넌트 | 구현 방식 |
|---------|---------|

| 체크박스 항목 | ☑ 심볼 + 경고 텍스트는 빨간 언더라인 |
